## 📝 Example Snippet

```javascript
// Normal Function
function greet(name) {
  return `Hello, ${name}!`;
}

// Arrow Function
const greetArrow = (name) => `Hello, ${name}!`;

console.log(greet("Sayan"));       // Output: Hello, Sayan!
console.log(greetArrow("World"));  // Output: Hello, World!
